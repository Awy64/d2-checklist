Wildcard Filter Info

The wildcard filter will filter by item name, along with masterwork, mod and perk names. It also has a few special commands (some of which match DIM's filters)

  • is:wishlist or godroll Show only items that match a god roll from DarkElement1987's curated list
  • is:masterwork Has a fully upgraded masterwork socket
  • is:hasmod Has a mod applied (so be careful infusing this into something else, since you'll lose the mod
  • random Is a random roll, and thus cannot be restored from Collections
  • fixed Is a fixed roll, you can always grab it from collections
